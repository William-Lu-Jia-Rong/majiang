# 雀账台

线下麻将实时记分台。

## 局域网使用

```bash
npm install
npm run lan
```

启动后终端会显示局域网地址，例如 `http://192.168.1.23:4173`。同一 Wi-Fi 下的手机或电脑打开这个地址，就会进入同一张牌桌。

默认大屏/电脑会进入主机展示页，只显示总分、庄家、最新事件、访问地址和玩家个人页二维码。

玩家手机扫码后会打开 `?view=player` 个人记分页，选择自己后只看自己的分数、排名、是不是庄和最近变化。

总控页负责开局、录入胡牌/给分/修正、重新指定庄家。可以打开：

```text
http://你的局域网IP:4173/?view=controller
```
